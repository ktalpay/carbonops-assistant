# CarbonOps Assistant Baseline Knowledge

## Current deterministic support
CarbonOps Assistant currently supports deterministic guardrails, response contract validation, and small helper utilities for units and emissions calculations.

## Inputs deterministic emissions helpers need
Deterministic emissions calculations need explicit activity data values, units, and an emissions factor with clear units.

## Limitations and non-goals
This baseline does not provide regulatory claims, certification, complete emissions accounting coverage, or external data ingestion.

## Why assumptions matter
Assumptions document boundaries, factor provenance, and calculation context so results are reviewable and repeatable.
