# v0.1.0 Release Notes

## Summary

v0.1.0 is a documentation and deterministic baseline milestone for CarbonOps Assistant. It records the first public release-candidate shape after the implemented package baseline, documented contracts, public examples, and test-only continuous integration.

This release includes:

- Python package baseline
- deterministic response contract
- guardrails baseline
- emissions calculation helpers
- unit normalization helpers
- sample question fixture
- sample question schema validation
- parser input contract
- source metadata model
- reporting result contract
- public question examples
- test-only CI

This release is not a production implementation, not a regulatory reporting tool, and not carbon accounting certification.

## Implemented deterministic baseline

- response statuses: `answered`, `unsupported`, `needs_more_context`
- guardrail checks
- emissions calculation helpers
- unit normalization helpers
- sample question fixture schema validation
- deterministic local evaluation case loading
- local Markdown knowledge section loading/search baseline
- deterministic assistant orchestrator
- test-only CI

## What is intentionally not included

- emission factor parser implementation
- CSV/table parsing implementation
- model/provider integration
- external API calls
- web crawling
- PDF/OCR extraction
- production reporting pipeline
- automated regulatory filing
- legal/compliance assurance
- carbon accounting certification

## Public safety posture

- unsupported or risky requests are explicitly rejected or bounded
- outputs require human review
- source metadata and reporting contracts do not imply correctness
- calculated outputs are not final or certified results
