# v0.2.0 Release Notes

## Summary

v0.2.0 is a pre-alpha deterministic local demo milestone for CarbonOps Assistant. It extends the documentation and baseline work into a minimal local workflow that can be run without external services.

## Added

- Parser result serialization for JSON-friendly display.
- Parser demo command for narrow text factor snippets.
- Minimal reporting result model.
- Markdown summary renderer for local review output.
- Evaluation summary output for sample question status checks.
- Quickstart documentation for install, tests, local demos, and limitations.

## Intentionally Not Included

- Production reporting.
- CSV or table parser.
- PDF or OCR extraction.
- External provider integration.
- External data retrieval.
- Regulatory filing.
- Certification.

## Notes

- Deterministic local demo milestone.
- Pre-alpha.
- Parser and reporting outputs remain candidates requiring validation and human review.
- The project does not claim carbon accounting correctness.
