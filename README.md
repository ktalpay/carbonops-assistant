# CarbonOps Assistant

CarbonOps Assistant is an early-stage, pre-alpha project for building a domain-specific assistant around carbon, emissions, reporting workflows, and technical evaluation.

## Status

Pre-alpha. Not production-ready.

## Current Scope

The initial goal is to create a small, testable assistant foundation with:

- domain-specific question handling
- emissions-related calculation helpers
- structured response contracts
- evaluation cases
- documentation and reproducible examples

## Non-Goals

This project does not currently provide:

- regulatory compliance certification
- production deployment
- legal or financial advice
- automated regulatory filing
- complete emissions accounting coverage
